כן: yes :: Basic confirmation. Example: כן, אני בא. — Yes, I’m coming.
לא: no :: Basic negation. Example: לא, תודה. — No, thanks.
תודה: thank you :: Used everywhere, formal and informal. Example: תודה על העזרה. — Thanks for the help.
בבקשה: please / you’re welcome :: Means both “please” and “you’re welcome.” Example: בבקשה תעזור לי. — Please help me.
סליחה: sorry / excuse me :: Used to apologize or get attention. Example: סליחה, איפה השירותים? — Excuse me, where are the bathroom?
מה: what :: Basic question word. Example: מה זה? — What is this?
מי: who :: Basic question word. Example: מי זה? — Who is that?
איפה: where :: Basic question word. Example: איפה אתה גר? — Where do you live?
מתי: when :: Basic question word. Example: מתי ניפגש? — When will we meet?
למה: why :: Basic question word. Example: למה אתה עייף? — Why are you tired?
איך: how :: Basic question word. Example: איך מגיעים לשם? — How do you get there?
אני: I :: First-person singular. Example: אני רעב. — I’m hungry.
אתה: you :: Masculine singular “you.” Example: אתה בבית? — Are you at home?
את: you :: Feminine singular “you.” Example: את מוכנה? — Are you ready?
הוא: he :: Third-person masculine. Example: הוא חבר שלי. — He is my friend.
היא: she :: Third-person feminine. Example: היא עובדת פה. — She works here.
אנחנו: we :: First-person plural. Example: אנחנו הולכים לים. — We’re going to the beach.
עכשיו: now :: Very common in speech. Example: אני יוצא עכשיו. — I’m leaving now.
היום: today :: Basic time word. Example: היום חם מאוד. — Today is very hot.
מחר: tomorrow :: Basic time word. Example: מחר יש לי פגישה. — Tomorrow I have a meeting.
אתמול: yesterday :: Basic time word. Example: אתמול הייתי בתל אביב. — Yesterday I was in Tel Aviv.
טוב: good :: Very common adjective. Example: זה רעיון טוב. — That’s a good idea.
רע: bad :: Opposite of טוב. Example: זה לא רע. — That’s not bad.
גדול: big :: Masculine form. Feminine: גדולה. Example: הבית גדול. — The house is big.
קטן: small :: Masculine form. Feminine: קטנה. Example: החדר קטן. — The room is small.
חם: hot :: For weather, food, or temperature. Example: הקפה חם. — The coffee is hot.
קר: cold :: For weather, food, or temperature. Example: בחוץ קר. — It’s cold outside.
טעים: tasty :: Very useful for food. Example: האוכל טעים. — The food is tasty.
מים: water :: Plural-looking noun but usually used as singular meaning. Example: אני רוצה מים. — I want water.
אוכל: food :: Also can mean “eating” or “I eat” depending on context. Example: יש פה אוכל טוב. — There’s good food here.
בית: house / home :: Basic noun. Example: אני בבית. — I’m at home.
רחוב: street :: Useful for directions. Example: הרחוב הזה שקט. — This street is quiet.
כסף: money :: Basic everyday word. Example: אין לי כסף מזומן. — I don’t have cash.
זמן: time :: Basic abstract noun. Example: אין לי זמן עכשיו. — I don’t have time now.
עבודה: work / job :: Can mean the workplace or work in general. Example: אני בדרך לעבודה. — I’m on the way to work.
חבר: friend :: Masculine friend; feminine: חברה. Also can mean boyfriend. Example: הוא חבר טוב. — He is a good friend.
חברה: friend / girlfriend / company :: Feminine friend, girlfriend, or company depending on context. Example: היא חברה שלי. — She is my friend / girlfriend.
יאללה: come on / let’s go :: Very common slang from Arabic, used constantly. Example: יאללה, בוא נלך. — Come on, let’s go.
סבבה: cool / okay :: Very common casual word meaning fine, okay, cool. Example: סבבה, נדבר מחר. — Cool, we’ll talk tomorrow.
אחלה: great / excellent :: Casual positive word. Example: אחלה רעיון. — Great idea.
וואלה: really / wow / well :: Flexible slang word; can show surprise, agreement, or hesitation. Example: וואלה, לא ידעתי. — Really, I didn’t know.
מגניב: cool :: Casual word for something cool or interesting. Example: זה ממש מגניב. — That’s really cool.
באסה: bummer :: Slang for something disappointing. Example: איזה באסה, הפסדנו. — What a bummer, we lost.
חבל: too bad / shame :: Used when something is unfortunate. Example: חבל שלא באת. — Too bad you didn’t come.
כיף: fun :: Very common. Example: היה ממש כיף. — It was really fun.
אחי: bro / my brother :: Informal address, mostly masculine-coded but widely heard. Example: אחי, מה קורה? — Bro, what’s up?
מה קורה: what’s up :: Everyday greeting. Example: היי, מה קורה? — Hey, what’s up?
הכל טוב: all good :: Common answer to “what’s up?” Example: הכל טוב, תודה. — All good, thanks.
בקטנה: no big deal :: Slang meaning something is minor/easy. Example: זה בקטנה, אני אטפל בזה. — No big deal, I’ll handle it.
זורם: down / flexible / goes with it :: Slang meaning someone is easygoing or willing. Example: אתה זורם לקפה? — Are you down for coffee?
חופר: annoying / talks too much :: Slang; literally “digging.” Example: הוא קצת חופר. — He talks too much / he’s being annoying.
