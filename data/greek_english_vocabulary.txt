ναι: yes :: Basic affirmative answer. Example: Ναι, θέλω νερό = Yes, I want water.
όχι: no :: Basic negative answer. Example: Όχι, δεν θέλω = No, I do not want.
γεια: hi :: Informal greeting and goodbye. Example: Γεια, τι κάνεις = Hi, how are you.
ευχαριστώ: thank you :: Use to thank someone. Example: Ευχαριστώ για το νερό = Thank you for the water.
παρακαλώ: you’re welcome :: Means you are welcome and can also mean please. Example: Παρακαλώ, κάτσε = Please, sit.
συγγνώμη: sorry / excuse me :: Use for sorry or excuse me. Example: Συγγνώμη, πού είναι η τουαλέτα = Excuse me, where is the bathroom.
εντάξει: okay :: Means okay or all right. Example: Εντάξει, πάμε = Okay, let us go.
καλό: good :: Neuter form of good, often used for things. Example: Καλό φαγητό = Good food.
κακό: bad :: Neuter form of bad. Example: Κακό νερό = Bad water.
εδώ: here :: Means here, near the speaker. Example: Έλα εδώ = Come here.
εκεί: there :: Means there, away from the speaker. Example: Κάτσε εκεί = Sit there.
τώρα: now :: Means now, at this moment. Example: Θέλω φαγητό τώρα = I want food now.
μετά: later :: Means later or after. Example: Θα πάμε μετά = We will go later.
πάνω: up :: Means up or above. Example: Κοίτα πάνω = Look up.
κάτω: down :: Means down or below. Example: Κοίτα κάτω = Look down.
δεξιά: right :: Means right as a direction. Example: Πήγαινε δεξιά = Go right.
αριστερά: left :: Means left as a direction. Example: Πήγαινε αριστερά = Go left.
σταμάτα: stop :: Imperative form meaning stop. Example: Σταμάτα τώρα = Stop now.
έλα: come :: Imperative form meaning come. Example: Έλα εδώ = Come here.
πήγαινε: go :: Imperative form meaning go. Example: Πήγαινε εκεί = Go there.
κοίτα: look :: Imperative form meaning look. Example: Κοίτα αυτό = Look at this.
άκου: listen :: Imperative form meaning listen. Example: Άκου με = Listen to me.
κάτσε: sit :: Imperative form meaning sit or stay. Example: Κάτσε εδώ = Sit here.
σήκω: stand up :: Imperative form meaning stand up or get up. Example: Σήκω τώρα = Stand up now.
βοήθεια: help :: Use when asking for help. Example: Βοήθεια, παρακαλώ = Help, please.
νερό: water :: Common noun for water. Example: Θέλω νερό = I want water.
φαγητό: food :: Common noun for food. Example: Θέλω φαγητό = I want food.
τουαλέτα: bathroom :: Means toilet or bathroom. Example: Πού είναι η τουαλέτα = Where is the bathroom.
πόσο: how much :: Question word for how much or how many. Example: Πόσο κάνει = How much does it cost.
πού: where :: Question word for where. Example: Πού είσαι = Where are you.
πότε: when :: Question word for when. Example: Πότε πάμε = When are we going.
γιατί: why :: Question word meaning why, and sometimes because in answers. Example: Γιατί όχι = Why not.
τι: what :: Question word for what. Example: Τι θέλεις = What do you want.
ποιος: who :: Masculine question word for who. Example: Ποιος είναι = Who is he.
θέλω: I want :: First person verb meaning I want. Example: Θέλω καφέ = I want coffee.
δεν θέλω: I don’t want :: Negative phrase meaning I do not want. Example: Δεν θέλω φαγητό = I do not want food.
ξέρω: I know / know :: Means I know or know. Example: Ξέρω ελληνικά = I know Greek.
δεν ξέρω: I don’t know :: Means I do not know. Example: Δεν ξέρω πού είναι = I do not know where it is.
καταλαβαίνω: I understand :: Means I understand. Example: Καταλαβαίνω λίγο = I understand a little.
δεν καταλαβαίνω: I don’t understand :: Means I do not understand. Example: Δεν καταλαβαίνω ελληνικά = I do not understand Greek.
και: and :: Basic connector meaning and. Example: Νερό και φαγητό = Water and food.
ή: or :: Basic connector meaning or. Example: Τώρα ή μετά = Now or later.
αλλά: but :: Connector meaning but. Example: Θέλω, αλλά δεν μπορώ = I want to, but I cannot.
όμως: however :: Means however or but, often stronger than αλλά. Example: Θέλω να πάω, όμως δεν μπορώ = I want to go, however I cannot.
επειδή: because :: Connector meaning because, used before the reason. Example: Δεν πάω επειδή είμαι κουρασμένος = I am not going because I am tired.
οπότε: so :: Means so or therefore, often for practical consequence. Example: Βρέχει, οπότε μένω εδώ = It is raining, so I am staying here.
άρα: therefore :: Means therefore, used for logical conclusion. Example: Είναι αργά, άρα πάμε = It is late, therefore we go.
λοιπόν: so / well :: Discourse word meaning so, well, or then. Example: Λοιπόν, πάμε = Well, let us go.
τότε: then :: Means then, at that time or in that case. Example: Τότε δεν ήξερα = Then I did not know.
πριν: before :: Means before. Example: Πριν πάμε, θέλω νερό = Before we go, I want water.
όταν: when :: Connector meaning when. Example: Όταν έρθεις, θα πάμε = When you come, we will go.
ενώ: while / whereas :: Means while or whereas, often contrasts two facts. Example: Θέλω καφέ, ενώ αυτός θέλει νερό = I want coffee, whereas he wants water.
αν: if :: Means if. Example: Αν θέλεις, πάμε = If you want, we go.
αν και: although :: Means although. Example: Αν και είναι αργά, πάμε = Although it is late, we go.
παρόλο που: even though :: Means even though, usually for contrast. Example: Παρόλο που βρέχει, πάμε = Even though it is raining, we go.
επίσης: also :: Means also or as well. Example: Θέλω νερό, επίσης θέλω φαγητό = I want water, I also want food.
ακόμα: still / yet :: Means still or yet, and sometimes more. Example: Είμαι ακόμα εδώ = I am still here.
ακόμα και: even :: Means even, used for emphasis. Example: Ακόμα και τώρα θέλω να πάω = Even now I want to go.
μόνο: only :: Means only. Example: Θέλω μόνο νερό = I only want water.
κυρίως: mainly :: Means mainly or mostly. Example: Μιλάω κυρίως αγγλικά = I mainly speak English.
σχεδόν: almost :: Means almost. Example: Σχεδόν τελείωσα = I almost finished.
ίσως: maybe :: Means maybe or perhaps. Example: Ίσως πάμε μετά = Maybe we will go later.
μάλλον: probably :: Means probably or rather. Example: Μάλλον θα πάω = I will probably go.
σίγουρα: surely :: Means surely or definitely. Example: Σίγουρα θέλω = I definitely want it.
φυσικά: of course :: Means of course or naturally. Example: Φυσικά, πάμε = Of course, let us go.
δηλαδή: that is / I mean :: Means that is or I mean, used to clarify. Example: Θέλω κάτι απλό, δηλαδή νερό = I want something simple, that is water.
για παράδειγμα: for example :: Means for example. Example: Θέλω κάτι, για παράδειγμα νερό = I want something, for example water.
όπως: like / as :: Means like or as. Example: Όπως είπα = As I said.
έτσι: like this / so :: Means like this, this way, or so. Example: Κάν’ το έτσι = Do it like this.
έτσι κι αλλιώς: anyway :: Means anyway or in any case. Example: Έτσι κι αλλιώς θα πάω = I will go anyway.
τελικά: finally / in the end :: Means finally or in the end. Example: Τελικά πήγαμε = In the end we went.
γενικά: generally :: Means generally. Example: Γενικά μου αρέσει = Generally, I like it.
συνήθως: usually :: Means usually. Example: Συνήθως πίνω νερό = I usually drink water.
μερικές φορές: sometimes :: Means sometimes. Example: Μερικές φορές πάω εκεί = Sometimes I go there.
πάντα: always :: Means always. Example: Πάντα θέλω καφέ = I always want coffee.
ποτέ: never / when :: Means never. Do not confuse with πότε meaning when. Example: Ποτέ δεν πάω εκεί = I never go there.
ακόμη: still / yet :: Means still or yet, close to ακόμα. Example: Είναι ακόμη εδώ = He is still here.
μέχρι: until :: Means until or up to. Example: Μέχρι αύριο = Until tomorrow.
από: from :: Means from or by, depending on context. Example: Είμαι από το Ισραήλ = I am from Israel.
προς: toward / towards :: Means toward or to, usually direction. Example: Πάμε προς τα δεξιά = We are going toward the right.
με: with :: Means with. Example: Με εσένα = With you.
χωρίς: without :: Means without. Example: Χωρίς νερό = Without water.
για: for :: Means for, about, or in order to, depending on context. Example: Αυτό είναι για εσένα = This is for you.
σε: to / in / at :: Very common preposition meaning to, in, or at. Example: Σε ένα σπίτι = In a house.
από τότε: since then :: Means since then. Example: Από τότε μένω εδώ = Since then I have lived here.
μέχρι τότε: until then :: Means until then. Example: Μέχρι τότε, μένω εδώ = Until then, I am staying here.
γι’ αυτό: that’s why :: Means that is why or for this reason. Example: Γι’ αυτό δεν πάω = That is why I am not going.
παρ’ όλα αυτά: nevertheless :: Means nevertheless or despite all that. Example: Βρέχει, παρ’ όλα αυτά πάμε = It is raining, nevertheless we go.
από την άλλη: on the other hand :: Means on the other hand. Example: Θέλω να πάω, από την άλλη είμαι κουρασμένος = I want to go, on the other hand I am tired.
με άλλα λόγια: in other words :: Means in other words, used to rephrase. Example: Με άλλα λόγια, δεν θέλω = In other words, I do not want it.
