آه / אַה: כן :: תשובה פשוטה ל״כן״. Example: آه، أنا جاي. — כן, אני בא.
إيه / אִיה: כן :: עוד צורה נפוצה ל״כן״ בדיבור. Example: إيه، بعرف. — כן, אני יודע.
لا / לַא: לא :: שלילה בסיסית. Example: لا، مش هلأ. — לא, לא עכשיו.
شكراً / שֻכְּרַן / שוקרן: תודה :: מילה רגילה לתודה, מובנת בכל מקום. Example: شكراً كتير. — תודה רבה.
يعطيك العافية / יַעְטִיכּ אִל־עַאפְיֵה: תודה / כל הכבוד :: אומרים למישהו שעשה משהו בשבילך או עבד קשה. Example: يعطيك العافية عالقهوة. — תודה על הקפה.
الله يعافيك / אַלְלַה יְעַאפִיכּ: בשמחה / תודה בחזרה :: תשובה טבעית ל־يعطيك العافية. Example: الله يعافيك. — בשמחה / תודה.
مرحبا / מַרְחַבַּא: שלום :: ברכת שלום רגילה. Example: مرحبا، كيفك؟ — שלום, מה שלומך?
أهلاً / אַהְלַן: שלום / אהלן :: גם ברכת שלום וגם תשובה ל״שלום״. Example: أهلاً وسهلاً. — אהלן וברוך הבא.
باي / בַּאי: ביי :: פרידה יומיומית, כמו בעברית. Example: باي، بنشوفك بكرا. — ביי, נראה אותך מחר.
طيب / טַיֵּבּ: טוב / בסדר :: מילה שימושית להסכמה או לפתיחת תשובה. Example: طيب، خلينا نحكي. — טוב, בוא נדבר.
تمام / תַמַאם: סבבה / מצוין :: תשובה נפוצה מאוד כשאומרים שהכול בסדר. Example: كل إشي تمام. — הכול סבבה.
منيح / מְנִיח: טוב :: צורה מדוברת ל״טוב״; לנקבה אומרים منيحة / מְנִיחַה. Example: الوضع منيح. — המצב טוב.
مش منيح / מִש מְנִיח: לא טוב :: שלילה של ״טוב״. Example: اليوم مش منيح. — היום לא טוב.
كيفك؟ / כִּיפַכּ: מה שלומך? :: פנייה לזכר. לנקבה: كيفك؟ / כִּיפֵכּ. Example: مرحبا، كيفك؟ — שלום, מה שלומך?
شو؟ / שוּ: מה? :: מילת שאלה בסיסית. Example: شو بدك؟ — מה אתה רוצה?
مين؟ / מִין: מי? :: מילת שאלה בסיסית. Example: مين هاد؟ — מי זה?
وين؟ / וֵין: איפה? :: מילת שאלה שימושית מאוד. Example: وين البيت؟ — איפה הבית?
إمتى؟ / אֵימְתַא: מתי? :: שאלה על זמן. Example: إمتى جاي؟ — מתי אתה בא?
ليش؟ / לֵיש: למה? :: שאלה על סיבה. Example: ليش هيك؟ — למה ככה?
كيف؟ / כִּיף: איך? :: שאלה על דרך או מצב. Example: كيف بروح لهناك؟ — איך הולכים לשם?
أنا / אַנַא: אני :: גוף ראשון יחיד. Example: أنا من تل أبيب. — אני מתל אביב.
إنتَ / אִנְתַ: אתה :: פנייה לזכר. Example: إنتَ جاهز؟ — אתה מוכן?
إنتِ / אִנְתִי: את :: פנייה לנקבה. Example: إنتِ جاهزة؟ — את מוכנה?
هو / הֻוֵּ: הוא :: גוף שלישי זכר. Example: هو صاحبي. — הוא חבר שלי.
هي / הִיֵּ: היא :: גוף שלישי נקבה. Example: هي أختي. — היא אחותי.
إحنا / אִחְנַא: אנחנו :: גוף ראשון רבים. Example: إحنا هون. — אנחנו כאן.
إنتو / אִנְתוּ: אתם / אתן :: פנייה לרבים, גם לזכרים וגם לנקבות. Example: إنتو جايين؟ — אתם באים?
همّ / הֻם: הם :: גוף שלישי רבים. Example: همّ بالبيت. — הם בבית.
بدي / בִּדִּי: אני רוצה :: אחת המילים הכי שימושיות בדיבור יומיומי. Example: بدي قهوة. — אני רוצה קפה.
بدك؟ / בִּדַּכּ: אתה רוצה? :: פנייה לזכר. לנקבה: بدك؟ / בִּדֵּכּ. Example: بدك مي؟ — אתה רוצה מים?
ما بدي / מַא בִּדִּי: אני לא רוצה :: שלילה של ״אני רוצה״. Example: ما بدي آكل. — אני לא רוצה לאכול.
عندي / עִנְדִי: יש לי :: צורה בסיסית להגיד שיש לך משהו. Example: عندي وقت. — יש לי זמן.
ما عندي / מַא עִנְדִי: אין לי :: שלילה של ״יש לי״. Example: ما عندي مصاري. — אין לי כסף.
في / פִי: יש :: מילה בסיסית ל״יש״. Example: في أكل؟ — יש אוכל?
ما في / מַא פִי: אין :: שלילה של ״יש״. Example: ما في مشكلة. — אין בעיה.
هون / הוֹן: כאן / פה :: מציין מקום קרוב. Example: أنا هون. — אני פה.
غاد / עַ׳אד: שם :: מילה מדוברת ל״שם״. Example: السيارة غاد. — האוטו שם.
هلأ / הַלַּא: עכשיו :: מילה מאוד נפוצה ל״עכשיו״. Example: هلأ بدي أروح. — עכשיו אני רוצה ללכת.
اليوم / אִלְיוֹם: היום :: מילה בסיסית לזמן. Example: اليوم الجو حلو. — היום מזג האוויר יפה.
بكرا / בֻּכְּרַא: מחר :: מילה בסיסית לזמן. Example: بشوفك بكرا. — אראה אותך מחר.
مبارح / מְבַּארֵח: אתמול :: מילה בסיסית לזמן. Example: مبارح كنت هون. — אתמול הייתי פה.
مصاري / מַסַארִי: כסף :: מילה מדוברת מאוד ל״כסף״. Example: معي مصاري. — יש עליי כסף.
مي / מַי: מים :: מילה יומיומית מאוד. Example: بدي مي، لو سمحت. — אני רוצה מים, בבקשה.
أكل / אַכֵּל: אוכל :: שם עצם בסיסי. Example: الأكل زاكي. — האוכל טעים.
زاكي / זַאכִּי: טעים :: נפוץ מאוד לאוכל. Example: الشاورما زاكية. — השווארמה טעימה.
بيت / בֵּית: בית :: מילה בסיסית ושימושית. Example: أنا رايح عالبيت. — אני הולך הביתה.
شغل / שֻע׳וּל: עבודה :: משמש גם לעבודה וגם לדברים/עניינים, לפי ההקשר. Example: عندي شغل اليوم. — יש לי עבודה היום.
صاحب / סַאחֵבּ: חבר :: חבר זכר; חברה אומרים صاحبة / סַאחְבֵּה. Example: هو صاحبي. — הוא חבר שלי.
مبسوط / מַבְּסוּט: מרוצה / שמח :: כמו בעברית ״מבסוט״, אבל בערבית זו מילה רגילה לגמרי. Example: أنا مبسوط كتير. — אני ממש מרוצה.
زعلان / זַעְלַאן: עצוב / כועס :: יכול להיות עצוב, פגוע או כועס לפי ההקשר. Example: ليش زعلان؟ — למה אתה עצוב/כועס?
خلص / חַלַס: נגמר / די / סיימנו :: מילה שימושית מאוד; כמו ״חלאס״ בעברית. Example: خلص، بكفي. — די, מספיק.
عادي / עַאדִי: רגיל / לא נורא :: משמש כדי להגיד שמשהו בסדר או לא עניין גדול. Example: عادي، ولا يهمك. — לא נורא, אל תדאג.
مش مشكلة / מִש מֻשְכִּלֵה: אין בעיה :: ביטוי שימושי מאוד. Example: مش مشكلة، بساعدك. — אין בעיה, אעזור לך.
